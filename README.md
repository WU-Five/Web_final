# [107-2] Web Programming Final Project
### (Group 37) - pdf online discussion

### 一句話描述這個服務在做什麼
* 這是一個可以讓人上傳pdf並互相以螢幕錄影的方式分享討論的網站

### 安裝/使用/操作方式 (含伺服器端以及使用者端)
* Download  
    `git clone https://github.com/WU-Five/Web_final.git`
* Install  
    `npm install`  
    `cd client && npm install`
* Run  
    `npm run dev`
